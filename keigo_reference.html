01

敬語の基本分類

尊敬語
相手の行動・状態を高めて表現。主語が相手（目上の人）のときに使う。
「言う」→ おっしゃる
「行く」→ いらっしゃる
「見る」→ ご覧になる
「する」→ なさる
謙譲語
自分の行動をへりくだって表現。主語が自分（または身内）のときに使う。
「言う」→ 申し上げる
「行く」→ 伺う・参る
「見る」→ 拝見する
「する」→ いたす
丁寧語
「です」「ます」「ございます」を使って丁寧に表現する。美化語（お・ご）も含む。
「言う」→ 言います
「行く」→ 行きます
「見る」→ 見ます
名前 → お名前

※ 文化審議会答申「敬語の指針」(2007年) では5分類（謙譲語Ⅰ・Ⅱ、丁寧語・美化語）とされているが、本リファレンスでは実用的な3分類で整理する。

07

よくあるNG表現と正しい言い方

「なるほど」「確かに」
相手の言葉を自分が評価・判定しているニュアンスを含む。目上の人への返答として不適切。
✗ なるほどですね。確かにそうですね。
✓ おっしゃるとおりです。はい、さようでございます。
「参考になりました」
相手の言葉を自分の知識の「補足材料」と扱う表現。目上の人に対して謙譲の気持ちが足りない。
✗ 大変参考になりました。
✓ 大変勉強になりました。
「よろしかったでしょうか」
現在の確認事項を過去形で表す誤り。日本語として不自然。
✗ 次回選考日は○月○日でよろしかったでしょうか？
✓ 次回選考日は○月○日でよろしいでしょうか？
「〜になります」の誤用
「〜になります」は状態の変化・移行を表す。変化しない物や場所に使うのは誤り。
✗ 履歴書になります。
✓ 履歴書でございます。
✗ 自宅の最寄り駅は○○になります。
✓ 自宅の最寄り駅は○○です。
※「子どもは今年3歳になります」「就職して5年になります」など、変化・経過を表す場合は正しい。
「させていただく」の多用・誤用
「させていただく」は相手の許可を得て自分が行動し、そのことで恩恵を受ける場合に限り使う表現。一方的な行動には不適切。また繰り返し使うと回りくどく聞こえる。
✗ 現在の職場ではチームリーダーをさせていただいております。
✓ 現在の職場ではチームリーダーをしております。
✗ 希望させていただきます。（一方的な行動のため不可）
✓ 参加させていただきます。（相手の許可が伴う場合は可）
「ご苦労さまです」の誤用
「ご苦労さま」は目上の人が目下の人に使う労いの言葉。自分が目下の立場でこの言葉を使うのは失礼にあたる。
✗ （上司に対して）ご苦労さまでした。
✓ お疲れさまでした。
✓ 本日はお時間をいただきありがとうございました。
尊敬語・謙譲語の混同
主語が「相手」なら尊敬語、「自分」なら謙譲語。これを混同すると逆の意味になる。
✗ ○○様はおられますか？（「おる」は謙譲語）
✓ ○○様はいらっしゃいますか？
✗ 社長にお聞きになります。（自分の動作に尊敬語は不可）
✓ 社長に伺います。
✗ ご拝受いただければ幸いです。（「拝受」は謙譲語で相手に使えない）
✓ ご覧いただければ幸いです。
